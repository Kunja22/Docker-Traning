## 📖 Project Description

This project is a **Node.js application** designed for learning and demonstrating Docker containerization.  
The main goal of this project is to **run a Node.js application inside a Docker container** to simplify deployment and ensure consistency across different environments.

The application showcases a simple web interface and can be easily extended with additional features. By dockerizing the app, you can:

- Avoid environment conflicts between development and production.
- Quickly set up and deploy the application on any system with Docker installed.
- Share your application as a lightweight, portable Docker image.

### 🔹 Key Features

- Node.js backend application
- Dockerized setup for easy deployment
- Simple web interface to demonstrate running app
- Lightweight and portable container

---

### 🐳 Docker Integration

The application includes a **Dockerfile** that builds a Docker image of the Node.js app.  
Once the image is built, you can create and run a container to see the app running locally in a reproducible environment.
